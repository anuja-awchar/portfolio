# Anuja Awchar - Portfolio

Welcome to my personal portfolio website! I'm a Computer Science Engineering student and aspiring full-stack developer passionate about building modern web applications.

## 🚀 Features

- **Modern UI** - Sleek design with gradient backgrounds and smooth animations
- **Responsive** - Works perfectly on all devices
- **Dark Mode** - Toggle between light and dark themes
- **Project Showcase** - Displays my GitHub projects with live demos
- **Contact Form** - Get in touch easily

## 🛠️ Tech Stack

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- Font Awesome Icons
- Google Fonts

## 📂 Project Structure

```
portfolio/
├── index.html      # Main HTML file
├── styles.css      # Custom CSS styles
├── script.js       # JavaScript functionality
└── README.md       # This file
```
